Business System Network Design

Designed and implemented a redundant hierarchical network in Cisco Packet Tracer, incorporating two routers and two multilayer switches at each layer for redundancy, with connectivity to two ISPs. The network utilized VLANs for departmental segmentation, inter-VLAN routing on multilayer switches, and dynamic IP allocation via DHCP, with static IPs for server devices. Configurations included subnetting, OSPF routing, SSH for secure remote access, and port security in the Finance and Accounts department using sticky MAC and violation mode shutdown. Implemented NAT overload for internet access and ACLs for traffic control, along with WLANs for departmental wireless connectivity. The network was thoroughly tested to ensure seamless communication, optimal performance, and robust security while adhering to design requirements.

Bank System Network Design

Designed and implemented a hierarchical network topology using Cisco Packet Tracer, incorporating VLAN-based segmentation with subnetting and inter-VLAN routing for seamless communication between departments. The network included wired and wireless connectivity for approximately 60 users per department, dynamic IP allocation via DHCP servers, and static IP configuration for servers. Configured OSPF for route advertisement, SSH for secure remote access, and port security using sticky MAC with violation mode shutdown. End devices and servers were integrated with HTTP and email services. Basic device configurations, including hostnames, encrypted passwords, and banner messages, were applied, and domain IP lookup was disabled. Subnetting of the base address 192.168.10.0 was performed to allocate appropriate address ranges to each department. The network design was rigorously tested to ensure optimal performance, security, and communication across all devices.
